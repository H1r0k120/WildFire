# Wildfire Simulator (森林火災シミュレータ)

## 概要
JavaScript (p5.js) で動作する、セル・オートマトンをベースにしたインタラクティブな山火事延焼シミュレーターです。
森林密度、乾燥度（延焼確率）、風向・風速に加え、**地形の高低差（傾斜）による延焼速度の変化**をリアルタイムにシミュレートできます。また、本物の航空写真を読み込ませてその地形のままシミュレートする実験的なモードも搭載しています。

## 特徴・機能

- **インタラクティブな着火**: キャンバス上の任意の場所をクリックして、リアルタイムに火種（🔥）を設置できます。
- **動的なパラメータ変更**:
  - **森林密度**: 密であるほど火が燃え移りやすくなります。
  - **乾燥度（延焼確率）**: 基本的な火の勢いを調整します。
  - **風向・風速**: 風下（矢印の方向）への延焼確率が上昇します。
  - **傾斜の影響度**: 火は低い場所（暗い色）から高い場所（明るい色）へ向かって燃え広がりやすくなります。
- **リアルタイム統計モニター**: 経過ステップ数、生存している森林の割合、生存率を動的に数値で可視化します。
- **航空写真モード**: 手持ちの画像（地形図やサテライト写真など）をアップロードすると、画像の明るさを「標高」、色合いを「地質カテゴリ（森林・水・雪・岩など）」に自動変換してシミュレートできます。

## 🛠️ 技術スタック

- HTML5 / CSS3
- JavaScript (ES6+)
- [p5.js](https://p5js.org/) (グラフィック・描画ライブラリ)

## 実行方法
1. パラメータスライダーで環境条件を設定します。
2. シミュレートウィンドウの任意の場所をクリックして開始します。
